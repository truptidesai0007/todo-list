# Smart Todo List App

## Features

- Add tasks
- Delete tasks
- Complete tasks
- LocalStorage support
- Express API
- JSON file database
- Offline fallback
- Due dates
- Priority system
- Undo delete


## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
